## Website（前后端分离）

该目录采用“前端 Vue + 后端 Node/Express”的分离架构，参考：[`casa0017-web-assessment`](https://github.com/rorschachwilpeng/casa0017-web-assessment)。

### 目录结构
```
Website/
  front-end/   # Vue (Vite) + Axios + (可选) Router + Element Plus
  back-end/    # Node.js + Express：REST API + 本地 JSON 落盘
```
